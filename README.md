# ⚡ Electricity Bill Predictor

A simple web application that predicts electricity bills based on user input.

## 🚀 Features

* 📊 Slab-based billing system
* ✅ Input validation for correct data entry
* 💡 Instant bill calculation
* 🎨 Clean and responsive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## 📌 How to Use

1. Enter electricity units
2. Click "Calculate"
3. View your bill instantly

## 📈 Future Enhancements

* Add monthly usage tracking
* Graph visualization
* Save bill history
